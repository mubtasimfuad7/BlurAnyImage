# 📸 Blur Any Image

A professional, Material 3 Android application to apply high-quality real-time blur effects to your photos.

## ✨ Features
- **Modern UI**: Fully Material 3 design with smooth transitions.
- **Intense Blur Engine**: Custom-built blur logic (0-100) using RenderScript for background bitmap processing and RenderEffect for UI previews.
- **Theme Support**: Seamlessly switch between **Dark** and **Light** modes (Dark mode by default).
- **Touch to Compare**: Long-press the image to instantly view the original unblurred version.
- **Orientation Aware**: Automatically saves your work during screen rotations.
- **Privacy First**: Saves images to a dedicated `Pictures/BlurredImages` folder.

## 📲 Installation
You can download the ready-to-install APK from the `release/` folder in this repository.

1. Download `BlurAnyImage.apk`.
2. Open the file on your Android device.
3. Allow "Install from unknown sources" if prompted.
